# biocode

BIOCAD bioinformatics toolkit for phylogenetic analysis, mutation detection, and clade identification.

## Installation

```bash
pip install -e ./biocode
```

## Modules

- `biocode.trees` - Phylogenetic tree operations (IQ-TREE, MrBayes)
- `biocode.clades` - Confident clade identification
- `biocode.mutations` - Mutation analysis
- `biocode.model` - Data models
- `biocode.tools` - External tool management
- `biocode.config` - Configuration
- `biocode.errors` - Custom exceptions
